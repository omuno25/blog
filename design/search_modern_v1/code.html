AI Model Authentication

Explore the underlying security architecture used to authenticate requests between your client applications and our edge-based AI inference models. This guide covers tokens, session management, and rate limiting parameters.

security arrow_outward

Mutual TLS Authentication

The highest security tier for enterprise infrastructure, ensuring bidirectional verification between client and inference server.

key

API Keys

Standard bearer tokens for rapid prototyping and public client access.

auth.config.ts 
import { AIAuthProvider } from "@devdocs/ai-sdk";

export const auth = new AIAuthProvider({
  apiKey: process.env.DEVDOCS_API_KEY,
  region: "us-east-1",
  timeout: 5000
});
warning Important Note

Never expose your DEVDOCS_API_KEY on the client side. Use environment variables and server-side functions to handle all authenticated AI calls.
